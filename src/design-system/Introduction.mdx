# Momozzi 디자인 시스템

이 디자인 시스템은 Momozzi 앱의 UI 컴포넌트를 체계적으로 구성하고 관리하기 위해 만들어졌습니다.
Panda CSS와 함께 사용되어 일관된 디자인 언어를 구현합니다.

## 디자인 시스템 구조

디자인 시스템은 다음과 같은 구조로 이루어져 있습니다:

### 1. 기초 요소 (Foundations)

- **컬러 시스템**: 앱 전체에서 사용되는 색상 팔레트
- **타이포그래피**: 폰트 패밀리, 사이즈, 두께 등
- **간격 (Spacing)**: 여백과 간격에 관한 규칙
- **그림자 (Shadows)**: 입체감과 깊이를 위한 그림자 스타일

### 2. 컴포넌트 (Components)

- **버튼 (Button)**: 사용자 상호작용을 위한 기본 버튼
- **모달 (Modal)**: 팝업 형태로 콘텐츠를 표시
- **스피너 (LoadSpinner)**: 로딩 상태를 표시
- **여백 (VerticalSpace)**: 수직 여백을 제공

### 3. 레이아웃 (Layouts)

- 페이지의 구조와 콘텐츠 배치를 위한 레이아웃 컴포넌트

## 사용 방법

디자인 시스템의 컴포넌트는 다음과 같이 사용할 수 있습니다:

```jsx
import { Button } from '../design-system/components/buttons';
import { Modal } from '../design-system/components/modals';
import { LoadSpinner } from '../design-system/components/spinners';
import { VerticalSpace } from '../design-system/components/spacing';

// 사용 예시
<Button variant="primary" size="md" onClick={() => alert('Button clicked!')}>
  버튼 텍스트
</Button>

<VerticalSpace size={16} />

<Modal isOpen={isModalOpen} onClose={() => setIsModalOpen(false)} size="md">
  <div>모달 내용</div>
</Modal>
```

## 확장 및 개선

이 디자인 시스템은 계속해서 개선되고 확장될 예정입니다. 새로운 컴포넌트, 패턴, 스타일이 추가될 것이며,
기존 컴포넌트도 필요에 따라 업데이트될 것입니다.
