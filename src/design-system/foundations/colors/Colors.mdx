# 색상 시스템

Momozzi 앱의 시각적 아이덴티티를 구성하는 색상 시스템입니다. 
이 시스템은 일관된 사용자 경험을 제공하고, 브랜드 정체성을 강화하며, 
접근성을 보장하기 위해 설계되었습니다.

## 주요 색상 팔레트

| primary.300 | #66a5be | 중간 밝은 primary 색상 |

| primary.500 | #0F5777 | 기본 primary 색상 |

## 피드백 색상 팔레트

상태 표시와 피드백을 위한 색상 팔레트입니다.

| 이름 | 색상 코드 | 설명 |
|------|-----------|------|
| feedback.error | #EF4444 | 오류 상태 표시 |
| feedback.warning | #F59E0B | 경고 상태 표시 |
| feedback.success | #10B981 | 성공 상태 표시 |
| feedback.info | #3B82F6 | 정보 상태 표시 |

## 색상 사용 가이드라인

### 주요 색상 사용

- **primary.500**: 주요 브랜드 색상, CTA 버튼, 중요 요소 강조
- **primary.600 ~ 700**: 호버 상태, 활성화된 버튼 상태
- **primary.200 ~ 400**: 보조 강조, 아이콘, 부가 정보
- **primary.50 ~ 100**: 배경 강조, 선택된 상태 배경

### 중립 색상 사용

- **neutrals.white, neutrals.50**: 주요 배경 색상
- **neutrals.100 ~ 200**: 카드 배경, 경계선
- **neutrals.600**: 기본 텍스트 색상
- **neutrals.400 ~ 500**: 보조 텍스트, 비활성화된 요소
- **neutrals.700 ~ 900**: 제목, 강조 텍스트

### 피드백 색상 사용

- **feedback.error**: 오류 메시지, 삭제 버튼
- **feedback.warning**: 주의 메시지, 경고 표시
- **feedback.success**: 성공 메시지, 완료 표시
- **feedback.info**: 정보 메시지, 도움말

## 사용 예시

```jsx
import { css } from 'styled-system/css';

// 주요 버튼
<button className={css({
  background: 'primary.500',
  color: 'neutrals.white',
  '&:hover': {
    background: 'primary.600',
  }
})}>
  버튼 텍스트
</button>

// 보조 버튼
<button className={css({
  background: 'neutrals.50',
  color: 'neutrals.600',
  border: '1px solid',
  borderColor: 'neutrals.200',
})}>
  보조 버튼
</button>

// 오류 메시지
<div className={css({
  color: 'feedback.error',
  backgroundColor: 'rgba(239, 68, 68, 0.1)',
  padding: '2',
  borderRadius: '4px',
})}>
  오류가 발생했습니다.
</div>
```
